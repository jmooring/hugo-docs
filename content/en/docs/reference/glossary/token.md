---
title: token
---

A _token_ is an identifier within a format string, beginning with a colon and replaced with a value when rendered. Use tokens when:

  - Configuring [file caches][], [front matter][], and [permalinks][]
  - Localizing [dates][]
  - Setting the [`url`][] in front matter

  [`url`]: /docs/concepts/url-management/#tokens
  [dates]: /docs/reference/functions/time/format/#localization
  [file caches]: /docs/reference/configuration/caches/#tokens
  [front matter]: /docs/reference/configuration/front-matter/#tokens
  [permalinks]: /docs/reference/configuration/permalinks/#tokens
